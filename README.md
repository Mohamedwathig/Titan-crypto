
# Titan-crypto
A React-based cryptocurrency tracker using Redux Toolkit Query and Chart.js. Fetches real-time data and detailed coin stats from the CoinRanking API, including price history, market cap, supply, and live charts. Features interactive UI, responsive design with Ant Design, and a crypto news section powered by RapidAPI.
